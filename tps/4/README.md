# **Trabajo práctico** #

## **Parte 1** ##

## **Parte 2**  ##

Crear un programa que dada un variable arr, cree una nueva variable e invierta la posición de sus elementos. 
Debe cumplir con el siguiente comportamiento:

![](http://k30.kn3.net/2/8/6/C/2/9/087.png)
